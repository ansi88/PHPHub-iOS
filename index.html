介绍

PHPHub for iOS 是 PHPHub 的官方 iOS 客户端, 同时兼容 iPhone 和 iPad, 使用了最新的 PHPHub API, 目前该 API 仍在完善中, 待完全稳定之后会开放给所有开发者使用, 敬请期待. 若您在使用此应用的过程中有发现任何问题, 请到 GitHub 上给我提 issue 或 PR, 我将会不定期抽出时间来对一些 Bug 进行修复或更新.

安装

安装 Cocoapods

本项目使用 Cocoapods 来对第三方库进行管理, 如果你的电脑未安装过 Cocoapods, 运行以下命令进行安装.

$ gem install cocoapods

$ pod setup

安装第三方库.

$ pod install

配置项目

项目的一些较为隐秘的信息都放在了 SecretConstant.h 中, 因此你需要先创建自己的 SecretConstant.h 文件, 运行以下命令进行创建.

$ cp PHPHub/Constants/SecretConstant.example.h PHPHub/Constants/SecretConstant.h

SecretConstant 文件中的 Client_id 和 Client_secret 只能用于测试环境, 若要用在生产环境, 可以点击此链接进行申请.

由于本项目用到了 JPush 作为消息推送, 因此你需要为 JPush 创建 PushConfig.plist 文件, 运行以下命令进行创建.

$ cp PHPHub/PushConfig.example.plist PHPHub/PushConfig.plist

确保你将此项目运行在测试环境上.

打开 Xcode, 选择 Product -> Scheme -> Edit Schemes -> Info -> 更改 Build Configuration 为 Debug 模式.

打开 PHPHub.xcworkspace.

OK, 这就完了, 祝你玩得开心.

怎样登录?

测试环境

打开 PHPHub iOS 应用, 点击进入 "我的页面" 使用 "扫码登录" 来扫描下方二维码完成登录.

生产环境

去 PHPHub 论坛 使用你的 GitHub 账号进行登录, 完成之后进入个人资料页面, 扫描个人资料页面的客户端登录二维码完成登录, 如下图所示.
